# **App Name**: PDF Insights AI

## Core Features:

- PDF Upload: Allow users to upload PDF files.
- AI Question Answering: Use AI to answer questions based on the content of the uploaded PDF file, reasoning and deciding what information from the document to include in the response; function as a tool.
- User Interface: Provide a simple and intuitive user interface for interaction.

## Style Guidelines:

- Primary color: Soft blue (#89B8C2) to convey trust and intelligence.
- Background color: Light grey (#F0F4F5) to ensure readability and reduce eye strain.
- Accent color: Pale violet (#B18CB8) for interactive elements, standing out without overwhelming the interface.
- Body and headline font: 'Inter', a sans-serif font known for its clean, readable design, which is well-suited for both headlines and body text.
- Simple, professional icons to represent document actions.
- Clean and intuitive layout, prioritizing easy navigation and readability.
- Subtle transitions for uploading and displaying content, to give a smooth user experience.
